#### Button component:

```jsx
<Button>Login</Button>
```
#### Disabled Button component:

```jsx
<Button disabled={true}>Login</Button>
```

#### Success Button
```jsx
<Button mod="success">Login</Button>
```

#### Cancel Button
```jsx
<div style={{backgroundColor: 'gray', display: 'inline-block'}}>
   <Button type='reset' mod="cancel">Login</Button>
</div>
```

#### Navigation Button
```jsx
<Button mod="navigation">Login</Button>
```

#### GithubLogin Button
```jsx
<Button mod="githubLogin">Login</Button>
```
